# Workspace Artifacts

이 디렉토리는 리더와 워커가 공유하는 중간 산출물 저장소다.

## 파일명 규칙
- 형식: `{phase}_{agent}_{artifact}.md`
- 예시:
  - `01_lead_delivery_plan.md`
  - `03_design_wireframes.md`
  - `04_frontend_build_notes.md`
  - `05_qa_release_report.md`

## 사용 규칙
- 채팅 메시지보다 `_workspace/` 파일을 우선한다.
- 다음 워커가 읽어야 하는 내용은 반드시 파일로 남긴다.
- 최종본과 중간 메모를 섞지 않는다.
- 완료 후에도 감사 추적이 가능하도록 명령, 결정, 위험을 남긴다.
